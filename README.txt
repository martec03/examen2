PROYECTO DE DESARROLLO DE UN SOFTWARE ADMINISTRACION DE LA AGENDA DEL MAE, BAJO LA PLATAFORMA WEB

El proyecto consiste de desarrollar un software o aplicacion para que el o la persona responables
pueda administrar la agenda de la MAE (Maxima Autoridad Ejecutiva) de una Institucion publica.

Ademas, la aplicacion debe ser al 100% web, es decir bajo la plataforma de internet, donde la secretaria 
o el personal responsable registra segun la llegada de las correspondencias al despacho de la autoridad.

Por otro lado, la MAE ingresa al sistema evalua y actualiza el estado de cada cita o agenda, pudiendo ser (CANCALADO, CONFIRMADO, RECHAZADO).
